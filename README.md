# AI Visibility Methodology

**Developed by Demand Spring**

A proprietary four-layer diagnostic framework for industrial B2B companies to assess and close the gap between their technical expertise and their digital visibility — in search, trade publications, and AI answer engines.

## The Four Layers

1. **Findability Audit** — Organic search footprint, branded vs. non-branded traffic, competitive position
2. **Keyword Territory Mapping** — The full keyword universe buyers use, cross-referenced against current rankings
3. **Authority Signal Assessment** — Backlink quality against industry trade publications and associations
4. **AI Discoverability Analysis** — Content readiness for AI answer engines (Google AI Overviews, ChatGPT, Perplexity, Claude)

## This Repository

This repo contains the live React-based client deliverable template for the AI Visibility Methodology, deployed via Vercel.
